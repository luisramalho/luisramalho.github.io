--- layout: post title: How to get the Facebook Access Token, the Extended Page Access Token that never expires excerpt: "Explanation of how ti get an Extended Page Access Token that never expires" comments: true ---

You will need:

  • [APPLICATION_ID]
  • [APPLICATION_SECRET] (https://developers.facebook.com/apps/[APPLICATION_ID])
{% highlight bash %} https://www.facebook.com/dialog/oauth?client_id=[APPLICATION_ID] &scope=manage_pages,publish_stream &response_type=token &redirect_uri=https://www.facebook.com/connect/login_success.html {% endhighlight %}

When you see 'Success' then you have a URL like:

{% highlight bash %} https://www.facebook.com/connect/login_success.html#access_token=[ACCESS_TOKEN]&expires_in=[SECONDS] {% endhighlight %}

Grab the [ACCESS_TOKEN] then,

{% highlight bash %} https://graph.facebook.com/oauth/access_token?client_id=[APPLICATION_ID] &client_secret=[APPLICATION_SECRET] &grant_type=fb_exchange_token &fb_exchange_token=[ACCESS_TOKEN] {% endhighlight %}

Finally, you will see your extended page access token that never expires

{% highlight bash %} access_token=[EXTENDED_ACCESS_TOKEN] {% endhighlight %} {% highlight bash %} https://graph.facebook.com/me/accounts?access_token=[EXTENDED_ACCESS_TOKEN] {% endhighlight %}
